# IUCAA_Physics_of_Galaxy_Formation_Course

This repository contains Python-based Jupyter notebooks focused on semi-analytical and semi-numerical models of cosmic reionization and the 21-cm signal. The tutorials have been developed as part of the **Physics of Galaxy Formation** course, organised by IUCAA, held from June 8 - 24, 2026.

These tutorials are designed to be run using **Google Colab**. To work on Google Colab, you will need a **Google account**.

The tutorials are organized as follows:

- **Day 1:** Semi-analytical models of cosmic reionization.
- **Day 2:** Semi-numerical models of cosmic reionization ('SCRIPT') and the 21-cm signal.

A Jupyter notebook (`.ipynb` file) for each tutorial is available in this GitHub repository.

## Getting Started

1. **Download the notebook**
   - Download the required Jupyter notebook (`.ipynb` file) from this repository to your computer.

2. **Open Google Colab**
   - Go to: https://colab.research.google.com/

3. **Upload the notebook**
   - On the welcome screen, click the **Upload** tab and select your downloaded notebook file.
   - Alternatively, after Colab opens, navigate to **File → Upload notebook** and choose the `.ipynb` file.

4. **Start working**
   - Once uploaded, the notebook will open in a new Google Colab session.
   - You can view, edit, and execute the code cells directly in your browser.

5. **Run the code**
   - To execute a cell, click the **Run (▶)** button next to the cell, or use:
     - `Ctrl + Enter` : Run the current cell.
     - `Shift + Enter` : Run the current cell and move to the next cell.

## Notes

- Google Colab runs entirely in the cloud; no local Python installation is required.
- Some notebooks may prompt you to install additional Python packages during the session. Simply execute the provided installation cells when instructed.
